# CareSync – Medication & Appointment Management Web App

A mobile-first healthcare web app for elderly patients and caregivers.

## Features
- Patient & Caregiver login with OTP
- Medicine schedule with mark-as-taken
- Appointment calendar with reminders
- AI Health Assistant (Claude API)
- Browser push notifications for medicine reminders
- Accessibility: large text, high contrast, multilingual

## Live Demo
Open index.html in any browser. Works offline too.

## Tech
HTML · CSS · Vanilla JS · Claude AI API · localStorage
